关键交互流程

激活传送 首页 + 按钮 → 地图选点 → 填写信息 → 上传图片 → 保存 → 返回地图（新 Marker 已出现）
查看传送 地图点击 Marker / 抽屉点击卡片 → 详情页 → 展示信息+图片+备注 → 「传送」按钮导航
管理传送 我的页面 → 筛选分类 → 点击编辑（同一详情页切编辑模式）→ 修改/删除
